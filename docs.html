OpenKeyVal.org

OpenKeyVal.org is a completely open key-value data store, exposed as a drop-dead simple web service. The goal is to make this a very easy way to persist data in web applications.

The API

Store data by POSTing form data to http://openkeyval.org/some-arbitrary-key. The value posted as 'data' will be saved to some-arbitrary key. Returns the text 'ok' and an 200 status code on success.

Delete data by posting an empty string as 'data' to the same address. Returns a 200 status code and either 'removed' or 'not-found', depending on whether the value existed.

Fetch data by GETting http://openkeyval.org/some-arbitrary-key. Returns the data with a 200 status code on success, or a 404 if not found.

To fetch that data and serve it with an arbitrary MIME type: http://openkeyval.org/some-arbitrary-key.image/png

Valid keys

Keys can contain upper and lowercase characters, dashes, and underscores and must be at least 5 characters long.

Valid data

Data is completely freeform and is binary safe. Data size is currently limited to 64 KiB.

Examples

Curl

Post

$ curl -d "data=Meat+Dress" http://openkeyval.org/Attire

Get

$ curl http://openkeyval.org/Attire
